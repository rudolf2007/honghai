企業

業務連續性是最重要的
大規模的業務需要專用的負載平衡和備用設備去支援企業通信特大流量的需要。安全、政策遵從、業務連續性、記錄和報告都是至關重要的。

 

解決方案
高性能、可靠和備用是企業網絡的基本要求。Network Box 的 IT 安全解決方案能夠善用支援多重核心處理器、數兆位元組的 RAID 硬體和熱插拔重型電源供應器的硬件平台。為提供更大的彈性和性能表現，Network Box 系統可以安裝一個實時熱用備份，甚至多個Network Box 作負載平衡。

 

網絡保安管理服務
Network Box 的網絡保安管理服務能提供的遠遠超過其他只以安全作為服務（SAAS）供應商為多。除了具有其他競爭者提供的標準安全，Network Box 還提供：

  • 監測 – 由位於安全中心的安全分析家監測一千多個世界各地的網絡數據，為企業提供全天候的保護。
  • 威脅分析 – Network Box 所提供的資料都跟安全管理中心作出核對，這使得Network Box 的系統能針對本地及全球性的威脅作出實時應對。如果需要，自動更新功能會為所有客戶的系統作出實時更新以得到最新的保護水平。
  • 頻寬管理 – 通過分析數據流量、排除不必要的流量和管理關鍵業務流量，而提供最優質的流量服務。
  • 許可證管理 – 不需多個許可證：一個系統只需要一個許可證已可以涵蓋所有的用戶和安全應用。您可以節省不少時間在管理用戶，並相信每個用戶都受到保護。
  • 變更控制系統 – 我們的防火牆安全專家會按您的要求來驗證您的系統，以確保網路安全不受到損害。所有的更改都會保存到代碼管理數據庫和配置安全系統以作備份，這一切意味著Network Box 的客戶可依賴此系統來得到妥善的管理。
  • 負載平衡 – 每一個的網路連接都可單一自行配置其流量，使之適合您的業務需求。
  • 安全顧問 – 不論您有的是網絡安全結構，政策制定，脆弱性評估，檢驗或攻擊對應的擔心，我們的支援小組都可以為你解決問題。
